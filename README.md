# FoodHub - Restaurant Demand Analysis

## 📌 Project Overview
This project focuses on exploratory data analysis (EDA) for a food 
aggregator company. The goal was to understand customer demand 
patterns across different restaurants and cuisines, helping the 
business improve customer experience and make informed decisions.

## 🎯 Objective
- Understand which cuisines and restaurants are most in demand
- Analyze customer ordering behavior and delivery patterns
- Provide actionable business recommendations based on data insights

## 🛠️ Tools & Skills Used
- Python
- Pandas
- NumPy
- Seaborn
- Univariate Analysis
- Bivariate Analysis
- Exploratory Data Analysis (EDA)

## 🔍 Approach
1. Data cleaning and preprocessing to handle missing/inconsistent values
2. Univariate analysis to understand individual variable distributions 
   (e.g., cuisine type, ratings, cost, delivery time)
3. Bivariate analysis to explore relationships between variables 
   (e.g., cuisine vs. ratings, cost vs. order frequency)
4. Visualization of trends using Seaborn to identify patterns

## ✅ Business Recommendation
Based on the analysis, recommendations were provided to help the 
company focus marketing and operational efforts on high-demand 
cuisines and improve service in low-performing areas.
